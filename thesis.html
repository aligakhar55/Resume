Thesis Abstract

Abstract:

As the number of vehicles is increasing day by day for the last few years, but the roads and parking areas are not being constructed accordingly, there is a shortage of parking spaces in urban areas. The E-Parking System is the best solution to solve this problem.

This E-Parking system is implemented with the help of Internet of Things (IOTs) to replace the current conventional parking system. IOT is generally used so that two physical devices can exchange data/information. In this system, Arduino Uno is used, a microcontroller that can communicate, interact, and control physical devices.

The system uses RFID technology so the authorized users can enter the parking area using tags and cards. The user has an Android app to check the availability of parking slots before entering the parking area. Users must maintain a minimum balance to access the parking area; otherwise, the system will prevent them from entering.

Overall, by utilizing this technology, the system will aid the users in reducing human effort and time and help the owners of parking areas to manage them more effectively.
